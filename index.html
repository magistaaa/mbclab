MBC Update

Multimedia Application, Big Data & Cybersecurity Laboratory

Laboratorium Aplikasi Multimedia, Big Data dan Keamanan Siber yang berlokasi di Fakultas Teknik Elektro Universitas Telkom dan beroperasi di bawah Kelompok Keahlian Jaringan, Sibernetika dan Manajemen Rekayasa (NCM), merupakan fasilitas penelitian khusus dengan fokus multi-segi. Utamanya, bidang ini mempelajari bidang aplikasi multimedia, khususnya Teknologi Game dan Sistem Informasi Geografis (GIS), yang memanfaatkan teknologi mutakhir untuk mengeksplorasi simulasi interaktif dan analisis data spasial. Kami juga memiliki 4 divisi yaitu:

  • Big Data
  • Cyber Security
  • Geographic Information System
  • Game Tech

Divisi MBC Laboratory

Big Data

Divisi big data mempunyai 3 fokus yaitu Data Analisis, Data Scientist, dan Data Engineer. Pada intinya divisi big data memiliki tujuan untuk mengolah data agar mudah dipahami orang lain.

    Kegiatan divisi: Build Portofolio
    Risetan: Effiency energy
    Lomba yang diikuti: Kaggle Competition
    Target lomba yang diikuti: BRI Data Hackathon
Penjelasan Big Data

Cyber Security

Divisi cyber security berfokus pada pengamanan, riset, dan juga pembelajaran jaringan. Cyber security di MBC Laboratorium juga mendalami tentang dev ops seperti docker dan terraform.

    Kegiatan divisi: Persiapan CTF
    Risetan: Riset Smart Mosque
    Lomba yang diikuti: CTF
    Target lomba yang diikuti: GEMASTIK
Penjelasan Cyber Security

Geographic Information System

Divisi Geographic Information System atau yang lebih sering disebut GIS. Divi GIS mempunyai 3 fokus yaitu pengelolaan aplikasi mobile & UI/UX, permodelan karakter 3D & augmented reality, dan remote sensing.

    Kegiatan divisi: 3D Modelling
    Risetan: urban geography 3d modelling
    Lomba yang diikuti: PKM
    Target lomba yang diikuti: GEMASTIK
Penjelasan GIS

Game Tech

Divisi game tech berfokus pada pengembangan game yang telah dirancang sesuai tema & genre yang sudah dikonsepkan menggunakan software unity & adobe illustrator.

    Kegiatan divisi: Membuat game edukasi
    Risetan: education game
    Lomba yang diikuti: PUSPIPTEK MOGRAPH & PKM
    Target lomba yang diikuti: GEMASTIK
Penjelasan Game Tech
